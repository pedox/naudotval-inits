### npm

set global path prevent permission

```
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
nano ~/.bashrc

  export PATH=~/.npm-global/bin:$PATH
```

## yarn && windows-build-tools (Windows)

```
npm i -g yarn
yarn global add windows-build-tools --verbose
```
